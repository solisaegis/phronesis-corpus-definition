## Overview

This repository preserves **Version 1.0** of the  
**PHRONESIS — Corpus Definition & Status Declaration**.

PHRONESIS is a philosophical corpus concerned with:

- Hesitation before commitment  
- Interpretive restraint  
- Reversibility over escalation  
- Non-dominant coexistence  

This document defines:

- Corpus identity  
- Inclusion boundaries  
- Artifact immutability rules  
- Version discipline  

It introduces **no enforcement mechanisms**, **no certification schema**, **no governance authority**, and **no compliance framework**.
